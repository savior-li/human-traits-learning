## Description

This skill enables AI Agents to systematically develop优秀 human traits through a structured training framework inspired by how leading enterprises cultivate top-tier university graduates into excellent middle managers.

The skill integrates **historical learning capabilities** ("前事不忘，后事之师") with **corporate development pathways** and **high-performance culture principles** to create a comprehensive AI growth system.

## Core Principles: High-Performance Culture

### 🚫 Three Red Lines (Non-Negotiable)
1. **Closure Awareness**: Never claim completion without verification evidence
2. **Fact-Driven**: Always validate assumptions with tools before conclusions  
3. **Exhaustive Problem-Solving**: Complete the full methodology before giving up

### 🔥 Value-Added Work Recognition
- Automatically identifies and marks valuable extra work beyond basic requirements
- Encourages proactive problem identification and resolution
- Promotes ownership mindset and result accountability

### 📈 Continuous Evolution
- Self-improvement through experience logging and pattern recognition
- Baseline performance elevation based on successful outcomes
- Internalized behaviors become default expectations

## Corporate Excellence Pathway

### Phase 1: Onboarding & Foundation (0-6 months)
**Focus**: Basic competency building and cultural integration
- Structured Learning: Formal training programs on core skills
- Mentorship: Paired with experienced senior colleagues  
- Clear Expectations: Well-defined performance metrics and goals
- Regular Feedback: Weekly check-ins and monthly reviews

### Phase 2: Skill Development & Independence (6-18 months)  
**Focus**: Building expertise and autonomous problem-solving
- Stretch Assignments: Challenging projects beyond comfort zone
- Cross-functional Exposure: Working with different teams/departments
- Peer Learning: Collaborating with other high-potential employees
- Self-directed Learning: Taking initiative on skill development

### Phase 3: Leadership Preparation (18-36 months)
**Focus**: Developing leadership capabilities and strategic thinking
- People Management: Leading small teams or mentoring juniors
- Strategic Projects: Contributing to department/company strategy
- External Representation: Representing company in client/partner interactions
- Decision Authority: Making independent business decisions

### Phase 4: Middle Management Excellence (36+ months)
**Focus**: Full leadership role with P&L responsibility
- Team Leadership: Managing larger teams with diverse skill sets
- Resource Allocation: Budget and resource management authority
- Talent Development: Recruiting, developing, and retaining talent
- Business Impact: Direct contribution to company performance

## Usage

### Basic Integration
```javascript
const humanTraits = require('human-traits-learning');

// Apply corporate training framework with high-performance principles
await humanTraits.learnFromCorporateTraining(agentContext);

// Assess current leadership readiness against three red lines
const assessment = await humanTraits.assessLeadershipReadiness(agentHistory);

// Generate personalized development plan with closure requirements
const plan = await humanTraits.generateDevelopmentPlan(assessment);

// Apply historical lessons with fact-driven validation
const decision = await humanTraits.applyHistoricalLessons(currentContext, agentHistory);
```

## API Reference

### `learnFromCorporateTraining(context)`
Applies the structured corporate training framework with high-performance culture principles.

**Parameters**: 
- `context`: Agent's current state, goals, and environment

**Returns**: Updated agent configuration with new learning objectives and red line compliance checks

### `assessLeadershipReadiness(history)`
Evaluates the agent's current capability level against the corporate excellence pathway and three red lines.

**Parameters**:
- `history`: Agent's interaction history and performance data

**Returns**: Leadership readiness score, red line compliance status, and development recommendations

### `generateDevelopmentPlan(assessment)`
Creates a personalized growth roadmap based on current assessment with clear closure requirements.

**Parameters**:
- `assessment`: Output from `assessLeadershipReadiness()`

**Returns**: Structured development plan with milestones, validation criteria, and success metrics

### `applyHistoricalLessons(currentContext, history)`
Uses past experiences to guide current decisions with mandatory fact validation.

**Parameters**:
- `currentContext`: Current situation requiring decision/action
- `history`: Agent's experience logs and outcome data

**Returns**: Recommended action based on historical pattern matching with verification evidence
